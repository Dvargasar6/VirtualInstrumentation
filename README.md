# Virtual Instrumentation repository

This repository compiles the codes used to implement the practices of the subject Virtual Instrumentation for Physics Engineering at the National University of Colombia.
In this course, embedded systems were implemented using virtual instruments programmed in Python and ESP32 microcontrollers as a data acquisition card programmed in C++.

## Practice 1:

In this practice, a transmission system for modulated waves was implemented using an optical coupling between a infrared LED and a phototransistor. 
Virtual instruments were used for the generation of modulated waves, their demodulation, and the determination of their fundamental frequencies.


## Practice 2:

In this practice, a PID controller was implemented for two temperature and angular velocity systems.
